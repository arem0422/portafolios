# Portafolios · Slides

Sitio de una sola página con formato de diapositivas para presentar mi perfil profesional, experiencia, habilidades y proyectos destacados.

## Contenido
- Slide 1: Presentación personal y datos de contacto.
- Slide 2: Experiencia laboral reciente.
- Slide 3: Habilidades técnicas y formación.
- Slide 4: Grilla de 8 proyectos con imagen, cliente, stack y enlace.

## Stack
- HTML, CSS (custom), JavaScript vanilla.
- Tipografía Manrope desde Google Fonts.

## Cómo ver el sitio
1) Clona o descarga este repo.
2) Abre `index.html` en tu navegador.
- Navega con las flechas prev/next, los puntos o las teclas ← →.

## Personalización rápida
- Actualiza textos y datos en `index.html`.
- Sustituye las imágenes de proyectos en `/img/` y ajusta sus enlaces.
